### Unitech Banking App ###


Tech Stack:
  - Java 11, Gradle
  - Spring Boot -> Web, Security(JWT), Data JPA, Test, Validation
  - OpenFeign (for exchange rates service from `https://www.cbar.az/currency/rates`
  - PostgreSQL
  - Redis
  - Liquibase
  - DockerFile
  - Project Lombok
  - MapStruct
  - OpenApi (Swagger)
  - Slf4j
  - Render.com
  - Railway.app
